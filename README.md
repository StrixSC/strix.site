<img src="public/assets/logo.svg" alt="Logo Strix.Site" width="15%" align="right" />


## Stack

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [Tailwind CSS](https://tailwindcss.com/)

## Setup

1. Clone the repo
2. `cd strix.site/`
3. `npm install`
4. `npm run dev`
